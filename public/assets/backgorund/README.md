# Assets Folder

Folder ini berisi asset untuk track/lintasan balapan.

## Cara Menambahkan Asset Track

1. Simpan file gambar track dengan format berikut:
   - `track-1.png` - Track dasar
   - `track-bg.png` - Background lintasan

2. Ukuran yang disarankan: 800x600 pixel atau lebih besar
3. Format: PNG atau JPG

## Struktur File

```
/public/assets/tracks/
├── track-1.png
├── track-2.png
└── track-bg.png
```

# Assets Folder

Folder ini berisi asset untuk karakter pemain.

## Cara Menambahkan Asset Karakter

1. Simpan file gambar karakter dengan format berikut:
   - `racer-1.png`
   - `racer-2.png`
   - `racer-3.png`
   - `racer-4.png`
   - `racer-5.png`
   - `racer-6.png`

2. Ukuran yang disarankan: 200x200 pixel
3. Format: PNG dengan background transparan

## Struktur File

```
/public/assets/characters/
├── racer-1.png
├── racer-2.png
├── racer-3.png
├── racer-4.png
├── racer-5.png
└── racer-6.png
```
